# Editions

Each edition of the On Record instrument shares the same philosophy, page copy, paper, typography, trim, and binding. **Only the differences live here.** If something belongs to every edition, it stays in the shared top-level folders (`CONTENT/`, `PRINT/`, `RECORD/`, etc.) so the editions can never drift apart.

| Edition | What differs |
|---|---|
| [`man-on-record/`](man-on-record/) | Cover wordmark; cover colourway |
| [`woman-on-record/`](woman-on-record/) | Cover wordmark; cover colourway |
| [`the-common-record/`](the-common-record/) | Co-signed page copy, sealed-reveal mechanic, paired packaging, consent handling |

The two individual editions are **mechanically identical** — same instrument, same severity. Sameness is the point. The name simply lets the holder claim it in their own terms; it never implies a different *kind* of accountability.
