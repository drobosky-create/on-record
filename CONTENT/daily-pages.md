# Daily Page — Master

The repeating page, built as an InDesign master so every instance is identical. The Day count is the only numbering in the book. Deliberately austere: it asks whether the standard held and what was done — nothing more.

---

## Printed fields (in order)

**DAY ________**            ____ / ____ / ______

**Yesterday, the standards held:**   KEPT ☐    BROKEN ☐
_(If broken, one line — which, and why. No more.)_
_____________________________________________

**Today's single obligation:**
_____________________________________________

**The record** _(what was actually done):_
_____________________________________________
_____________________________________________

**Signed** ______________   _(close the day — no day is left unsigned)_

---

## Rules
- The daily page is **locked**. Designers must not add fields, prompts, ornament, or ruling beyond what is specified.
- KEPT / BROKEN is a binary mark — no scale, no partial credit.
- One obligation. The form physically refuses a list.
- Every day is signed. The signature line is the point of the page.
- Day count is the only numbering; no folio/page numbers.
