# Design Environment — InDesign is Non-Negotiable

## Decision
The Record is designed in **Adobe InDesign**. Full stop. Not because it is fancy — because it enforces discipline.

## Why
1. **It is a book, not a product.** Multi-page, print-first, signature-bound, layout- and gutter-dependent. InDesign is built for books, publishers, and long-form consistency.
2. **Absolute control** over baselines, leading, margins, gutter creep, section breaks, master pages, and page-numbering logic.
3. **It must feel administrative.** InDesign encourages restraint, repetition, and systematization. Marketer-oriented tools encourage variation, visual interest, and "delight." Delight is poison here.

## What not to use
- **Canva** — templated, decorative, poor print control.
- **Figma** — screen-first, no real print intelligence, weak long-document handling.
- **Google Docs / Word** — no professional control; printers dislike the files.

## If not designing it personally
Hire a book designer who works in InDesign. Ask for: "a book designer experienced with lay-flat binding and publisher-grade print files." Say **book** — not planner, journal, or brand design.

## Typography (decided)
**One typeface: IBM Plex Sans** — a neutral grotesque with an administrative, document pedigree. It reads as *form*, not literature, which fits the register exactly. If licensing/embedding for print is a problem, fall back to **Inter**.

- No second face, no display face, no pairing.
- Paragraph styles only — no local overrides.
- Confirm the licence permits **print embedding**.
