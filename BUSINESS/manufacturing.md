# Manufacturing

## Current phase
Seed Edition — validate the object, then a small seed run.

- **90-day edition** (see [`../PRINT/print-spec.md`](../PRINT/print-spec.md)).
- Run structure: **2–3 hand-bound proofs → ~50-unit seed run.** A true 12-unit sewn run is uneconomical at edition binderies.

## Process gates
1. Qualify 1–2 binderies against the print spec (genuine lay-flat is the hard requirement).
2. **Blank bound dummy** in exact stock + binding — confirm lay-flat, feel, bulk, spine, cover finish.
3. **Write test** — shipped pen on chosen stock.
4. **Foil/emboss draw-down** on actual cover material.
5. Lock the InDesign file and export per the file-prep checklist.
6. Approve a **full physical proof**.
7. Produce hand-bound proofs, then the ~50-unit seed run.
