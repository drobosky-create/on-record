# Man on Record

> Permanence changes behavior.

This repository is the single source of truth for the intellectual property, product design, manufacturing, and business behind **The Record** — and the platform it becomes.

The Record is Version 1. The company is an operating system for accountability.

## What lives here

| Folder | Contents |
|---|---|
| [`BRAND/`](BRAND/) | Vision, positioning, voice, and identity standards |
| [`CONTENT/`](CONTENT/) | The written content of The Record — identity standard, daily/weekly pages, initiation sequence |
| [`RECORD/`](RECORD/) | Production assets — InDesign files, print PDFs, fonts, cover art |
| [`PRINT/`](PRINT/) | Print specifications, paper stock, and vendor research |
| [`PACKAGING/`](PACKAGING/) | Box and pen specifications |
| [`OPA/`](OPA/) | Go-to-market strategy, seed partners, launch plan |
| [`BUSINESS/`](BUSINESS/) | Pricing, manufacturing, vendors |
| [`WEBSITE/`](WEBSITE/) | Web presence (future) |
| [`APP/`](APP/) | Digital companion / mobile app (future) |
| [`FUTURE-TOOLS/`](FUTURE-TOOLS/) | The Ledger, The Reckoning, The Archive |

## Working principle

Every meaningful change is **intentional, documented, and permanent** — the same discipline the product demands of its user. Use pull requests for significant changes, tag releases at each phase, and let the commit history carry the reasoning.
