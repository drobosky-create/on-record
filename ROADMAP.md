# Roadmap

Milestones map to release tags.

| Milestone | Tag | Exit criteria |
|---|---|---|
| Concept | `v0.1` | Brand, spec, and repo structure documented |
| Concept (production-real) | `v0.2` | 90-day extent, lay-flat binding, cloth + silver foil, single typeface |
| House + editions | `v0.3` | On Record house; Man / Woman / Common Record editions defined |
| Prototype | `v0.5` | Content final, InDesign file locked, printer qualified, physical proof approved |
| Seed Run | `v1.0` | Seed run produced (individual editions) and placed with seed partners |
| Common Record pilot | `v1.1` | Sealed-reveal mechanic proven; first paired sets tested with couples |
| Preorder | `v1.x` | Limited preorders open |
| Production | `v2.0` | First production run |

## Editions
- **Man on Record** / **Woman on Record** — individual 90-day editions (identical instrument).
- **The Common Record** — paired couple's edition; depends on the individual editions shipping first.
