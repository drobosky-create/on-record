# RECORD / Design — Claude Design package

Self-contained HTML design exports for The Record, delivered from Claude Design. Each file is standalone — open in any modern browser; no server or internet needed. These are **read-only visual deliverables**; the canonical written spec remains [`../design-brief.md`](../design-brief.md) and the shared [`../../CONTENT/`](../../CONTENT/) files.

## Files
- **Cover Spec.html** — cover production specification (colourways, foil, construction).
- **Editions.html** — the edition lineup (Man / Woman / The Common Record).
- **Interior Specs.html** — interior production spec + page layouts at 100%.
- **Interactive Models.html** — interactive page-through model of the book interior.

Received v0.3.x. If these ever diverge from the written spec, the written spec (`design-brief.md`, `CONTENT/`) is source of truth.
