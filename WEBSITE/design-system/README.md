# WEBSITE / design-system

The web & marketing brand system for On Record — the digital counterpart to the physical object's austerity. Same register: administrative, flat, hairline rules, sharp corners, oxblood only for action.

## Files
- [`style-guide.html`](style-guide.html) — live, self-contained style guide (open in any browser): colour, type, buttons, ruled form fields, surfaces, wordmark, principles.
- [`tokens.css`](tokens.css) — design tokens as CSS variables (the canonical values).
- [`ghl-brand-kit.md`](ghl-brand-kit.md) — paste-ready GoHighLevel setup: Brand Board values, fonts, and a Custom CSS block, with step-by-step.

## Source of truth
`tokens.css` holds the canonical values; the Figma library (Claude Design) mirrors them for designers. If anything diverges, the tokens win.
