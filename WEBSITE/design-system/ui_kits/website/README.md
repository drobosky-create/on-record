# UI Kit — Website

The On Record marketing/web surface (the register the GoHighLevel brand kit targets). One landing page composing the design-system primitives: ink hero with foil display line, edition lineup (three cloth covers set in type), the open/keep/close arc on ink cards, ruled waitlist form, ink footer.

- `index.html` — entry; renders `Landing.jsx`.
- Copy is drawn from BRAND/ and the repo README — declarative, administrative, no exclamation.
- No imagery, no icons, no shadows, no rounding — per the style guide's Don't list.
