The typographic wordmark — there is no drawn logo; always set the brand name in type.

```jsx
<div style={{ background: "var(--or-ink)", padding: 56, textAlign: "center" }}>
  <Wordmark foil />
</div>
<Wordmark edition="Man on Record" size={24} />
```

IBM Plex Sans 600, letter-spacing .02em. `foil` applies the silver-foil gradient — the ONLY gradient the brand permits, and only on dark surfaces. On light, solid ink. Wordmark only — never add a tagline lockup.
