# Launch Plan

## Phases
1. **Prototype** — finalize content + InDesign file; qualify printer; approve physical proof.
2. **Seed Run** — 12 units; place with seed partners; collect impressions.
3. **Preorder** — open limited preorders once the object is validated.
4. **Production** — first real production run.

Map each phase to a release tag: `v0.5` Prototype, `v1.0` Seed Edition, `v1.x` Preorder, `v2.0` Production.
